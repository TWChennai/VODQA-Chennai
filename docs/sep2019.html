Voice-based home assistants use conversational interfaces unlike other applications to interact. This session will cover the strategy for testing such interfaces, testing the NLP models and sharing experience on how to automate these tests and add it to the CI/CD build pipelines.

Element classification with ML in Appium

Appium enables test automation of mobile apps using Test.ai's machine-learning element type classifier used find Appium elements using a semantic label instead of having to dig through your app hierarchy. The same labels can be used to find elements with the same general shape across different apps and different visual designs.
